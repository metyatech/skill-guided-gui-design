# Contributing

## Scope

Keep this repository focused on reusable GUI guidance for user-facing products.

## Editing rules

- Keep `SKILL.md` concise and action-oriented
- Put persistent universal behavior in shared rules, not in this repo
- Keep examples generic and platform-agnostic
- Update `README.md` and `CHANGELOG.md` when behavior changes

## Verification

Run:

```sh
npm install
npm run verify
```
